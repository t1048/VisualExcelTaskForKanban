# VisualExcelTaskForKanban
Excelファイルのタスク一覧からカンバン形式で可視化するソース

## 優先度列の初期化／マイグレーション手順

既存の Excel データに新しい「優先度」列を追加する場合は、次の手順で準備してください。

1. カンバンで利用している Excel ファイル（既定では `task.xlsx`）を Excel などで開きます。
2. `担当者` 列の右隣に新しい列を挿入し、列名を **優先度** に変更します。
   - 既に列が存在する場合は名前だけを「優先度」に変更します。
3. 既存タスクの優先度を必要に応じて入力します。数値（1, 2, 3 など）でも「高／中／低」などのラベルでも構いません。
4. ファイルを上書き保存します。

※ バックエンドは列が存在しない場合でも自動的に追加しますが、手動で列を用意しておくと Excel 上の見通しが良くなります。
